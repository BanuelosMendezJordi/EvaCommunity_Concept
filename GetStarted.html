Get Started

For beginners, we developed a visual guide to help you understand and get access to the physical and logical components needed.

For advanced users and developers, you can directly access to the Github repository containing the documentation and instructions.

Eva Versions

Before starting to build your project, we should keep in mind that there is currently more than one version, so it's important to properly identify which version is the most optimal for your needs and purposes.

Basic

The most basic version of the Eva social robot has three key capabilities:

  • Verbal interaction using a microphone array and a speaker.
  • Eyes displaying through a 5,5" touchscreen
  • Head movement using two Dynamixel AX-12A servomotors.
