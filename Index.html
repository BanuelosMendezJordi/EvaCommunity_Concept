EVA

An open-source social robotics platform

About Eva

EVA (which stands for "Embodied Voice Assistant") consists in an open-source robotics platform, which provides most of the resources to build a fully functional social robot at an affordable cost. It has been designed to create interactions for specific contexts and populations.

Eva's repository provides all the elements to make your own social robot, such as 3D models, software, diagrams, and guidelines to assemble it using open-hardware solutions such as the Raspberry Pi and Arduino-based card-boards.
